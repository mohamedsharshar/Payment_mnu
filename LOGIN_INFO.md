# معلومات تسجيل الدخول - نظام الدفع الجامعي

## حسابات التجربة

### حساب الأدمن (Admin)
- **البريد الإلكتروني:** `admin@admin.com`
- **كلمة المرور:** `password`
- **الصلاحيات:** وصول كامل لجميع أقسام لوحة التحكم

### حساب الطالب (Student)
- **البريد الإلكتروني:** `student@student.com`
- **كلمة المرور:** `password`
- **الصلاحيات:** لا يمكنه الوصول للوحة التحكم

---

## نظام الصلاحيات (Spatie Permissions)

تم تثبيت وإعداد **Spatie Laravel Permission** بنجاح:

### الأدوار (Roles)
1. **admin** - مدير النظام
2. **student** - طالب عادي

### الصلاحيات (Permissions)
- `view-dashboard` - عرض لوحة التحكم
- `manage-customers` - إدارة الطلاب
- `manage-bills` - إدارة الفواتير
- `manage-services` - إدارة الخدمات
- `manage-payments` - إدارة المدفوعات
- `view-reports` - عرض التقارير

### الحماية
- جميع صفحات Admin Panel محمية بـ Middleware: `auth` و `admin`
- فقط المستخدمين الذين لديهم دور **admin** يمكنهم الوصول
- الطلاب العاديين يتم رفض وصولهم تلقائياً (403 Forbidden)

---

## ملاحظات

- عند تسجيل الخروج، يتم توجيهك لصفحة تسجيل الدخول
- إذا حاول طالب عادي تسجيل الدخول، سيتم رفض دخوله وعرض رسالة خطأ
- زر تسجيل الخروج موجود في أسفل القائمة الجانبية باللون الأحمر

---

## تشغيل النظام

```bash
# تشغيل الـ migrations مع الـ seeders
php artisan migrate:fresh --seed

# تشغيل السيرفر
php artisan serve
```

ثم افتح المتصفح على: http://localhost:8000/login
